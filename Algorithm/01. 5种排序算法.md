# 排序算法

## 加1减1

### 插入排序
```java
public class InsertionSort implements Sorter {
    @Override
    public void sort(int[] A){
        for(int i=0;i<A.length;i++){
           int c = A[i];
            for (j = i; j > 0 && A[j - 1] > c; j --) {
                A[j] = A[j - 1];
            }
            A[j] = c;
        }
    }
}
```
