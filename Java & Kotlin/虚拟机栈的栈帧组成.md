## 局部变量表

> 局部变量表按定义顺序存储两块内容

1. 存储方法参数
2. 存储方法内的局部变量

<a name="Emaht"></a>

### 局部变量表的特点

1. 线程私有,不允许跨线程访问,随方法调用创建,方法退出销毁
2. 编译期间长度已确定,局部变量元数据存储在字节码中
3. 局部变量表是栈帧最主要的存储空间,决定了栈的深度 <a name="u6ZQj"></a>
